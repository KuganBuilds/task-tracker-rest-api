## 🔄 Project Evolution

This project is an upgraded version of my earlier **CLI-based Task Tracker application**, which was built using core Java.

### Original CLI Version
The initial version focused on:
- Command-line interaction
- Core Java programming concepts
- Business logic implementation

### Spring Boot Upgrade
This version upgrades the same domain into a **production-style backend REST API** by:
- Implementing layered architecture (Controller, Service, Repository, DTO, Entity)
- Integrating MySQL using Spring Data JPA and Hibernate
- Exposing RESTful APIs with proper HTTP status codes
- Adding validation, global exception handling, and Swagger/OpenAPI documentation
